About

Elder Dragon Bot is a Bot for Discord where users are able to quickly access information regarding profile statistics, skins, champion builds and more right from Discord, removing the need to open a third-party application or website. The primary focus of Elder Dragon Bot is to provide the most common build paths for a champion in a way that is easy for the user to read, displaying skill level up order in a table-like format, as well as providing the most common rune pages in a way that the user should already be familiar with, that is, providing an imitation of the rune pages that are used within the League of Legends Client itself.

Commands

Command Syntax Description
~!build ~!build <champion> Displays common build paths
for specified champion
~!chests ~!chests <region> <summonername> Displays list of champions that have
not been granted a chest
~!commands ~!commands
~!commands <command>		 Displays list of commands
Displays example of command being used
~!ingame ~!ingame <region> <summonername> Displays live game information
~!prefix ~!prefix <custom prefix>
<custom prefix>prefix reset		 Changes the '~!' prefix to a custom prefix
Resets prefix to '~!'
~!profile ~!profile <region> <summonername> Displays profile information
~!skins ~!skins <champion> Displays list of skins for champion
React to view skin
~!tft ~!tft cheatsheet classes/origin/items
~!tft cheatsheet ultimate
~!tft cheatsheet chance
~!tft champion <champion>		 Displays cheatsheet
Displays for both items and classes/origins
Displays probability of tiers to roll
Displays champion details
~!topplayers ~!topplayers <region>
<queuetype(solo/duo, flex, tft)>		 Displays top ranked players in
specified region and queue
